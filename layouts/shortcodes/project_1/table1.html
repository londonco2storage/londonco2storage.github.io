Table 1. Summary of Definitions for Performance Metrics
Performance Metric Definitions
Capture rate capacity
  1. Maximum CO2 captured in a particular year
  2. Maximum amount of CO2 that can be captured in a year based on the facility design
  3. Average capture rate for a given period
  4. Intended capture target
Capture rate An estimate of the annual amount of CO2 that has been captured after the project commenced
Storage rate - hybrid An estimate that uses the annual storage rate where possible (only some projects provided this data) and the average storage rate
